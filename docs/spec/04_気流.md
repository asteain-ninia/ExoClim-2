風帯（Step 2）に加えて、陸海温度差・地形による圧力 anomaly と、それに伴う地衡風・季節風（モンスーン）を導出する。

## 3. 入力

> Phase 2 で確定。暫定列挙:

- 風帯（Step 2 出力）
- 気温（Step 5 出力との順序関係に注意。Pasta では気温と気流が相互依存する可能性あり）
- 海流（Step 3 出力。海面温度を介して影響）
- 地形（陸海分布・標高）

## 4. アルゴリズム概要

> Phase 2 で Pasta 該当節を抜粋して再構成する。

- モンスーン（季節風）の表現
- 地衡風 vs 圧力勾配風（赤道帯）の切替
