# Step 5: 気温

> **状態**: 骨格のみ。本文は Phase 2 で Pasta 一次資料から起こす。

## 1. 目的

惑星表面の月別気温分布を、日射・アルベド・温室効果・熱輸送・標高から導出する。

## 2. 一次参照

> Phase 2 で WebFetch して該当節を抽出する。

- **Climate Explorations: Temperature** (2022-05) — <https://worldbuildingpasta.blogspot.com/2022/05/climate-explorations-temperature.html>（一次）
- Worldbuilding Pasta — "An Apple Pie from Scratch, Part VIb" Step 4（Surface temperature）
- 補助: Hot & Cold Planet Climates

## 3. 入力

> Phase 2 で確定。暫定列挙:

- 月別TOA日射（Step 0.5 相当。軌道幾何）
- 大気パラメータ（温室効果・アルベド・lapse rate）
- 海洋パラメータ（熱容量・熱輸送効率）
- 海流（Step 3 出力。海面温度の偏差を作る）
- 地形（標高・陸海分布）

## 4. アルゴリズム概要

> Phase 2 で Pasta 該当節を抜粋して再構成する。

論点:
- 放射平衡（Stefan-Boltzmann）の扱い
- 海氷フィードバック（アルベド変化）
- 子午面熱輸送
- 海洋熱慣性（季節遅延）
- 標高補正（lapse rate）
- 海流による熱輸送（東岸暖流・西岸寒流）

## 5. 出力

> Phase 2 で確定。暫定:

- 月別の表面気温
- グローバル平均気温
- 最高/最低気温
