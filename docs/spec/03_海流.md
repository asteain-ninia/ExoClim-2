# Step 3: 海流

> **状態**: 骨格のみ。本文は Phase 2 で Pasta 一次資料から起こす。

## 1. 目的

惑星表面の海流（赤道反流・赤道流・中緯度循環・暖流/寒流）を、風帯と陸海分布から導出する。

## 3. 入力

> Phase 2 で確定。暫定列挙:

- 風帯（Step 2 出力）
- ITCZ 緯度線（Step 1 出力）
- 地形（陸海分布・大陸棚距離）

## 4. アルゴリズム概要

> Phase 2 で Pasta 該当節を抜粋して再構成する。

論点（旧版で議論された範囲）:
- 赤道反流（東向き）→ 赤道流（西向き）→ 中緯度海流 の三段階
- 大陸東岸での衝突 → 極方向への分岐
- 大陸西岸での衝突 → 戻り
- 暖流/寒流の判定基準
